# План перевода TheReanimator с немецкого на русский

## Краткое описание

Этот план содержит **атомарные шаги** для перевода приложения TheReanimator (Proxmox Management Suite) с немецкого языка на русский.

**Философия перевода**: Краткий, креативный русский перевод (не дословный). Англицизмы допустимы, но использовать их сдерживая - там, где можно обойтись без них, обходимся.

---

## ЧАСТЬ 1: УПРАВЛЕНИЕ ВЕТКАМИ

### Шаг 1: Создание ветки Development
**Действие**: Создать и переключиться на новую ветку

```bash
git checkout -b Development
git push -u origin Development
```

**Проверка**:
```bash
git branch  # Должно показать * Development
```

---

### Шаг 2: Процесс работы
1. Работаем исключительно в ветке `Development`
2. Тщательно тестируем переводы
3. Создаём Pull Request в `main` после завершения
4. Ждём одобрения пользователя
5. Делаем merge после одобрения
